  • features
    • set debug on/off
    • set background color by yourself
    • auto bind to all textarea (wont bind to hidden -- display:none, cause WYSIWYG may use them)
  • notice
    • still have few bugs in p and dd such commands
  • v0.1 -- reimplement
    • Common: ESC, N
    • Left-right motions: [N]h,[N]l,0,^,[N]$
    • Up-down motions: [N]k,[N]j,[N]-,[N]+,[N]_,[N]G,[N]gg,[N]%
    • Inserting text: [N]a,[N]A,[N]i,[N]I,[N]o,[N]O
    • Deleting text: [N]x,[N]X,[N]dd,[N]D,[N]J
    • Copying and moving text: [N]p,[N]P(only work with Deleting text, still have some bugs)
